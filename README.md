# Personal Blog

Astroで構築した個人技術ブログです。

## 技術スタック

- **Framework**: [Astro](https://astro.build/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Hosting**: GitHub Pages
- **Analytics**: Google Analytics 4
